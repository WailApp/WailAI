WailAI - روبوت دردشة

WailAI هو روبوت دردشة بسيط مصمم باستخدام HTML, CSS, وJavaScript.

التثبيت

لاستنساخ وتشغيل هذا المشروع، ستحتاج إلى Node.js وnpm مثبتين على جهاز الكمبيوتر الخاص بك.

1. استنساخ هذا المستودع:
`git clone https://github.com/WailAI/WailAI.git`

2. انتقل إلى مجلد المشروع:
`cd WailAI`

3. تثبيت التبعيات:
`npm install`

4. بدء التطبيق:
`npm start`

الاستخدام

بمجرد بدء التطبيق، افتح متصفح الويب الخاص بك وانتقل إلى http://localhost:3000 للاستمتاع بروبوت الدردشة.
